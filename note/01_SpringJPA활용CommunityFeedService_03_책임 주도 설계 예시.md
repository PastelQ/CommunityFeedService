# 책임 주도 설계 `과외 수강생 관리 프로그램`
## Use Case 1(예시). 새로운 수강생을 등록할 수 있어야 함
- 수강생에 필요한 정보는 나이, 이름, 성별 주소가 있음
- 기본적인 정보가 입력되지 않았을 때에는 에러를 반환함

> 행동 기반으로 추출
1. 사용자의 정보 입력
2. 유효성 검증
3. 데이터 저장
4. 결과 반환

## 책임 부여 단계별
1. 처음부터 행동 기반의 갯수나 **정확하게 추출될 필요는 없음**
   - 추후 구체화 중 맞춰갈 수 있다!
